# Git Tools

For efficient and guided creation of branches and commits in the terminal, please download the following tool:

## Recommended Tool

- **Repository:** https://github.com/T-vK/git-tools/tree/master

## Benefits

- Guided branch creation with naming conventions
- Automated commit message formatting
- Ensures adherence to Angular Commit Convention
- Simplifies common Git workflows

## Usage

The tool includes commands like:
- `mkbranch` - Interactive branch creation
- `mkcommit` - Guided commit creation with convention enforcement
